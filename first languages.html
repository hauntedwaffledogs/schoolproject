First languages

The first programming language was the assembly language which was first developed in the 1940s. In 1954 the coding language named FORTRAN was invented at IBM by a team led by John Backus. FORTRAN was used as a high-level general language to have a functional implementation. A nether coding language was called UNIVAC I which was invented by grace hopper in 1955.
